Cara Menggunakan Repository

Ikuti langkah-langkah berikut untuk mengonfigurasi sistem Linux Anda

Setup Ubuntu 24.04 LTS (Noble)

1

Backup sources.list

sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup
2

Ganti isi file sources.list

/etc/apt/sources.list
Ubuntu 24.04
deb http://repo.stembayo.sch.id/ubuntu/ noble main restricted universe multiverse
deb http://repo.stembayo.sch.id/ubuntu/ noble-updates main restricted universe multiverse
deb http://repo.stembayo.sch.id/ubuntu/ noble-security main restricted universe multiverse

Konfigurasi ini mengarahkan sistem Anda untuk mengunduh package langsung dari mirror lokal.

3

Update informasi paket

sudo apt update
Repository berhasil dikonfigurasi

Setup Debian 12 (Bookworm)

1

Backup sources.list

sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup
2

Ganti isi file sources.list

/etc/apt/sources.list
Debian 12
deb http://repo.stembayo.sch.id/debian/ bookworm main contrib non-free non-free-firmware
deb http://repo.stembayo.sch.id/debian/ bookworm-updates main contrib non-free non-free-firmware
deb http://repo.stembayo.sch.id/debian-security/ bookworm-security main contrib non-free non-free-firmware

Konfigurasi ini mencakup repository utama, updates, dan security Debian.

3

Update informasi paket

sudo apt update
Repository berhasil dikonfigurasi

Pertanyaan Umum

Pertanyaan yang sering diajukan tentang repository mirror

Repository mirror adalah salinan lokal dari repository paket Ubuntu atau Debian yang memungkinkan Anda mengunduh dan memperbarui paket dengan lebih cepat dari server lokal tanpa perlu menggunakan bandwidth internet. Mirror menyimpan salinan lengkap dari repository resmi dan diperbarui secara berkala.

Mirror repository kami disinkronisasi dengan repositori resmi setiap 10 jam untuk memastikan Anda memiliki akses ke pembaruan keamanan dan paket terbaru. Ini menjamin bahwa sistem Anda akan selalu mendapatkan patch keamanan penting tanpa penundaan yang signifikan.

Untuk kembali ke repository resmi, Anda dapat mengembalikan file sources.list dari backup yang Anda buat sebelumnya dengan perintah berikut:

sudo cp /etc/apt/sources.list.backup /etc/apt/sources.list
sudo apt update
